Initiative

戦闘時の先制判定を担う重要な要素です。ユニットのAGI、クラス回避補正、防具など様々な要素から値を算出し、高い方が先に攻撃。攻撃回数分繰り返し、1回攻撃する度にInitiativeが少しずつ下がっていきます。

撃破数増加量について

相手に止めを刺すと撃破数が増えます。その際、相手の最大HPに応じて増加量が変わります。

回復魔法

ヒーリング・ヒーリングオール・リザレクションが該当します。

普段は別の武器ですが、戦略・応戦中かつ後の条件を満たすと装備欄に反映、亡命・内乱コマンドが消え戦闘コマンドが変化します。ヒーリング・ヒーリングオールは回復魔法行使クラス、リザレクションは蘇生魔法行使クラスであることが条件です。

タロットカード

攻撃系カードで撃破してもON狩りにはなりません。

戦闘

対戦相手一覧の一番右に出るアルファベットは？

熟練度です。相手の強さの目安となりますが・・・嘘もあります。E以上は情報でステータスを見るといいかも。

戦闘後にユニットがステータスダメージ(STD)を受けると元に戻せないのですか？

STR、VIT、AGI、DEXのステータスが戦闘後ダメージを受けると、ユニットをレベルアップさせてまた成長させない限り元には戻りません。 レベルはレベルアップで、HPやMPなどは鍛練コマンドによって増やす事は出来ます。

ユニット名やコードネームを変えたいのですが。

ユニット名はログインIDとしての意味もある為変更は出来ません。コードネームはクラスチェンジによって変更する事は可能です。

亡命や内乱をした場合、階級はどうなりますか？

亡命は成功すると亡命先での階級は傭兵となります。失敗した場合も若干降格します。内乱は成功すると総帥や隊長に、失敗すれば傭兵に降格となります。

ステータスの増え方に法則はありますか？

現在のクラス補正を中心として上がっていきます。

一向に階級が上がりません。

何度か相手を撃破すると階級は上がりますが、逆に撃破されたときは階級が下がります。またリアルタイムで戦闘している為知らない間に誰かから攻撃を受け撃破されてしまったりするとその分階級が下がっています。

戦闘後、「攻撃力がわずかに上昇」という表示が出たのにランクが変わっていません。

ステータスのランクは数値をランクとして表示しているので、上昇しても表示が変わらない事がありますが、実際には数値は上がっています。何度か上昇するとランク表示も変わります。

熟練度の意味は？

熟練度はあくまでも他のプレイヤーから見た時のそのユニットの強さの目安です。熟練度が高くてもユニットの能力が上がるわけではありません。

戦闘時の作戦はどうすれば増えますか？

レベルの上昇と共に次第に増えていきます。

ログインすると戦闘もしていないのに回復中になっています。

戦闘はすべてリアルタイムで進行します。貴方がログインしていない時でも、他のプレイヤーからは対戦相手として戦闘が可能です。LAST BATTLEの欄に最後に行なった戦闘が表示されるので確認してください。

階級の上下関係

傭兵 → 二等兵 → 一等兵 → 上等兵 → 兵長 → 伍長 → 軍曹 → 曹長 → 二等准尉 → 一等准尉 → 少尉 → 中尉 → 大尉 → 少佐 → 中佐 → 大佐 → 准将 → 少将 → 中将 → 大将 → 上級大将 → 元帥

同じ人に狙われてばかりで行動できません。

フラグです。

レベルを上げてもステータスが上がらない

クラスチェンジした回数によって、ステータスの上限があります。 クラスチェンジが出来るようになったら、すぐにクラスチェンジしましょう。クラスチェンジは熟練度がDになってから10体撃破し、20000Goth所持していると出来るようになります。熟練度は100体撃破すると1ランク上がります。

倒せる相手を狙っているとすぐにALIがChaosになってしまいます。ALIを維持するには？

レベルの高い相手と戦い、あらかじめ自分のALIをある程度上げておいてから、 レベルとALIの低い相手を倒すとALIが下がり難いです。

休戦中に攻撃されているのにgothや武器経験値が増えません。

自分から戦闘をしかけないまま一両日ほど経過すると、休戦中に攻撃されてもお金や武器経験値がもらえなくなります。したがってログインだけしてゲームを終了しているとお金や武器経験値がもらえませんので、ログイン後に戦闘するようにしましょう。

国・部隊

部隊作成や建国には階級などの制限はありますか？

部隊作成は大佐以上の階級が必要です。建国には階級制限はありません。 資金さえあればいつでも建国できます。

国費はどのようにして増やすのでしょうか？

国籍がある場合、戦闘終了時に自動で国費が増えています。獲得量は相手の階級が高さに応じて。

一度作った部隊は消えないのでしょうか？

部隊の解散は総帥が特殊コマンドから行なわない限り無くなりません。

国籍があったのにいつのまにか無国籍になっています。

貴方の所属していた国が滅亡している可能性があります。 または総帥であった場合は誰か別の人物に内乱を起こされた可能性も考えられます。どちらの場合も歴史に記録されるので確認してみてください。

国に属することで何か良い点はありますか？

階級が上がるごとに貰える経験値とgothが増えます。傭兵時の撃破、被撃破による経験値とgothは無国籍にいる状態と変わりありません。

武器

武器の売却価格はどれくらいですか？

購入時の半額です。武器レベルが高くても売却価格は変わりません。

装備のレベルはどうやって上げるんですか？

戦闘をすれば、装備中の武器に経験値が入ります。一定の経験値がたまればレベルが上がります。

武器レベルの影響は？

わずかに攻撃力が上がります。命中率に影響はありません。指輪、盾は性能が上がります。

攻撃レベルと攻撃回数とが書いてありますが攻撃レベル×攻撃回数なのでしょうか？

違います。攻撃力 Ｘ 攻撃回数＝攻撃レベルになっています。10000 X 1 = ランクＡならば、2500 X 4 もランクＡということです。
